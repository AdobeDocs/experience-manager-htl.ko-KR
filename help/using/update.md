# HTL이란? {#what-is-htl}

**이전 명칭 *Sightly***

업데이트 날짜: 2016년 8월

HTML 템플릿 언어(HTL이라고도 함)는 HTML에 대한 Adobe Experience Manager의 기본 서버측 템플릿 시스템입니다. 모든 HTML 서버측 템플릿 시스템과 마찬가지로 HTL 파일은 HTML 자체와 런타임 시 평가할 몇 가지 기본 프레젠테이션 논리 및 변수를 지정하여 브라우저로 전송되는 출력을 정의합니다.

AEM 6.0 및 6.1에서 HTL은 **Sightly**&#x200B;라고 불렸었습니다. Adobe는 무엇을 위한 사양인지 명확히 하고 Adobe의 명명 지침에 맞추기 위해 “HTML 템플릿 언어”로 명칭을 변경했습니다. 이런 명칭 변경은 2016년 8월부터 적용되었으며 AEM 버전 6.0 이상에 적용됩니다.

>[!NOTE]
>
>이런 명칭 변경은 코드나 API에 영향을 미치지 않으므로 호환성에는 영향을 미치지 않습니다. 자세한 내용은 [이 공지 비디오를 참조하십시오.](https://helpx.adobe.com/kr/experience-manager/how-to/announce-htl.html)

HTL에 대한 자세한 정보와 좋은 시작 지점은 [HTML 템플릿 언어(HTL) 시작 안내서](overview.md)를 참조하십시오.
